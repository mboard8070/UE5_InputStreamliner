# Input Streamliner

An Unreal Engine 5 editor plugin that dramatically simplifies multiplatform input configuration using AI-powered natural language understanding.

## Overview

Input Streamliner allows developers to generate complete input systems—including Enhanced Input assets, platform-specific bindings, mobile touch controls, and runtime rebinding UI—in minutes rather than hours.

### Key Features

- **Natural Language Input Definition** - Describe your input needs conversationally and receive a complete, correctly-configured input system
- **Multiplatform Support** - PC (Keyboard/Mouse), Gamepad, Mac, iOS, Android
- **Mobile Touch Controls** - Virtual joysticks, buttons, D-pads, radial menus, gesture detection, gyroscope
- **Runtime Rebinding** - Generated settings UI for player input customization
- **C++ or Blueprint** - Choose your preferred binding approach
- **Dynamic Management** - Add, remove, and update input actions at any time

## Target Platforms

| Platform | Input Types |
|----------|-------------|
| PC | Keyboard/Mouse, Gamepad (Xbox, PlayStation, Generic) |
| Mac | Keyboard/Mouse, Gamepad |
| iOS | Touch, Gyroscope, MFi Controllers |
| Android | Touch, Gyroscope, Bluetooth Controllers |

## Requirements

- Unreal Engine 5.3+
- [Ollama](https://ollama.ai/) with Nemotron 8B model (for AI features)
